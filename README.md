# rick-and-morty-app
In this app you can find information about the characters from the Rick and Morty series.

## Requirements
- iOS 13+
- Xcode 13
- Swift 5

## This project uses:
- UIKit
- VIPER
- View Code
- URL Session
- Dependency Injection
